Issue: Dashboard shows blank page
Solution:
  1. Check browser console (F12) for errors
  2. Verify faculty_survey_data.csv is in the same folder
  3. Wait 2-3 minutes for GitHub Pages to fully deploy
  4. Clear browser cache (Ctrl + Shift + R)

Issue: "404 Not Found" error
Solution:
  1. Check URL spelling: /faculty-survey-dashboard/ (with dash, all lowercase)
  2. Verify files were uploaded to correct folder
  3. Check GitHub Pages is enabled in repository settings

Issue: Charts don't load
Solution:
  1. Check internet connection (CDN libraries required)
  2. Verify all three files are present: index.html, dashboard.js, CSV
  3. Check console for CORS errors

Issue: Data doesn't display
Solution:
  1. Ensure faculty_survey_data.csv is in the same folder as index.html
  2. File name must be exact: faculty_survey_data.csv (with underscore)
  3. Check CSV is not corrupted (open in Excel/Notepad)
