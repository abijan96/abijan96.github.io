# LDS Training Impact Dashboard

Interactive dashboard for analyzing Leadership Development Series (LDS) training evaluation survey results.

## Features

- **Interactive Visualizations**: KPI cards, bar charts, pie charts, histograms, and stacked bar charts
- **Dynamic Filtering**: Filter by training session and department
- **Excel Export**: Download editable Excel report with data and charts
- **File Upload**: Upload modified data to update dashboard in real-time
- **Real-time Updates**: All charts update based on filters and uploaded data

## Data Sheets in Excel Export

1. **Survey Data (Editable)** - Main data table for editing
2. **Summary Statistics** - Overall metrics and session breakdowns
3. **Department Analysis** - Department breakdown with charts
4. **Session Comparison Chart** - Visual comparison of sessions
5. **Likert Responses** - Response distribution
6. **Comments** - Participant feedback

## How to Use

### View Dashboard
Simply open the dashboard to view all training survey results and insights.

### Download & Edit Data
1. Click "Download Excel Report"
2. Open the Excel file
3. Edit data in the "Survey Data (Editable)" sheet
4. Save your changes

### Upload Updated Data
1. Click "Upload Updated Data"
2. Choose your modified Excel file or CSV file
3. Dashboard automatically refreshes with new data

## Technologies

- D3.js for data visualization
- SheetJS (xlsx) for Excel file parsing
- Vanilla JavaScript for interactivity
- Responsive CSS design
